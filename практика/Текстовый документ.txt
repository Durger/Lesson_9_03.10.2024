от 10 задач по работе с числовыми массивами, которые подойдут для начинающих студентов:

1. Сумма элементов массива:
   Напишите программу, которая принимает массив чисел и выводит сумму всех его элементов.

2. Среднее значение:
   Напишите программу, которая находит среднее значение элементов массива.

3. Максимальный и минимальный элементы:
   Напишите программу, которая находит максимальный и минимальный элементы в массиве.

4. Обратный порядок:
   Напишите программу, которая переворачивает массив (т.е. выводит его в обратном порядке).

5. Поиск элемента:
   Напишите программу, которая ищет заданный элемент в массиве и возвращает его индекс. Если элемент не найден, вывести сообщение об этом.

6. Удаление элемента:
   Напишите программу, которая удаляет заданный элемент из массива и выводит новый массив.

7. Сортировка массива:
   Реализуйте сортировку массива по возрастанию (например, методом пузырька).

8. Уникальные элементы:
   Напишите программу, которая находит и выводит уникальные элементы в массиве (т.е. элементы, которые встречаются только один раз).

9. Четные и нечетные числа:
   Напишите программу, которая разделяет массив на два: один с четными числами, другой — с нечетными.

10. Копирование массива:
    Напишите программу, которая копирует содержимое одного массива в другой.
